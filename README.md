## ✨ Возможности

### 📆 Календарь

- 🗓️ **Месячный вид** с навигацией по неделям
- 📝 **Еженедельные заметки** (опционально)
- 🎨 **Цветовые индикаторы** задач и привычек на ячейках календаря
- 🔢 **Бейджи** с количеством незавершённых задач
- 🔥 **Индикатор серии** (streak) для ежедневных активностей
- 📊 **Подсчёт слов** как визуальный метрик
- 🌍 **Поддержка локали** (русский язык) и настройка дня начала недели
- 💎 **Glassmorphism-дизайн** с адаптивной вёрсткой

### 🕐 Расписание (FullCalendar)

- 📅 **Недельный, дневной и месячный виды**
- ⏰ **Time Grid** с отображением задач по времени (06:00 - 00:00)
- 🖱️ **Drag & Drop** для переноса и изменения длительности задач
- 📱 **Swipe-навигация** на мобильных устройствах
- 🎨 **Тёмные цвета проектов** на календаре
- ➕ **Создание задач** по клику на пустое время
- ⏱️ **Форматирование времени**: минуты (< 60) или часы (≥ 60)

### ✅ Трекер задач

- 📋 **Три статуса**: Сделать → В работе → Готово (Kanban-стиль)
- 🏷️ **Проекты** с тёмными цветовой маркировкой
- ⚡ **Приоритеты**: низкий, средний, высокий
- 🔁 **Повторяющиеся задачи** (ежедневно, еженедельно, ежемесячно)
- 📅 **Автоматическая генерация** задач до конца месяца
- 🔗 **Привязка задач** к заметкам в хранилище
- ⏱️ **Таймер** отслеживания времени (plan → fact)
- 📈 **Оценка времени** и сравнение с фактическими затратами
- ⏰ **Запланированное время** с индикатором просроченности
- 🗄️ **Архивация заметок-задач** при завершении
- 📊 **Логи времени** с графиками и историей
- 🔍 **Поиск папок** при выборе пути

### 🎯 Трекер привычек

- 📅 **Ежедневные, еженедельные** и пользовательские частоты
- 🎨 **Иконки и цвета** для каждой привычки
- 🔥 **Подсчёт серии** (streak)
- 📊 **Отслеживание выполнения** по датам
- 🔗 **Привязка к проектам**
- 🧹 **Автоматическая очистка** истории (до 180 записей)

### 📊 Аналитика

- 📈 **График активности по неделям** (последние 12 недель)
- 📊 **Логи задач по времени** с визуализацией
- 📋 **Детальная статистика** по каждой привычке
- 🔥 **Текущая и лучшая серия**
- 📈 **Процент выполнения** за последние 30 дней

### 🔔 Уведомления

- ⏰ **Напоминания** о задачах с запланированным временем
- 🚨 **Оповещения** о просроченных задачах
- ⚙️ **Настройка времени** напоминаний (5, 10, 15, 30, 60 минут)

### 🔄 Синхронизация

- 💾 Сохранение данных в `calendar-data.json`
- ☁️ Синхронизация через **Obsidian Sync/Remotely Save**
- 🔁 Автоматическая перезагрузка при изменениях файла

## 📦 Установка

### Способ 1: Через Obsidian BRAT (рекомендуется)

1. Установите плагин [BRAT](https://github.com/TfTHacker/obsidian42-brat)
2. Откройте настройки BRAT → **Add Beta Plugin**
3. Вставьте URL: `https://github.com/AtinsS/obsidian-calendar-plugin-remastered`
4. Нажмите **Add Plugin**

### Способ 2: Ручная установка

1. Скачайте `main.js`, `manifest.json` и `styles.css` из [последнего релиза](https://github.com/AtinsS/obsidian-calendar-plugin-remastered/releases)
2. Создайте папку `.obsidian/plugins/calendar/` в вашем хранилище
3. Скопируйте скачанные файлы в эту папку
4. Перезапустите Obsidian
5. Включите плагин в **Настройки → Сторонние плагины**

### Способ 3: Из исходного кода

```bash
git clone https://github.com/AtinsS/obsidian-calendar-plugin-remastered.git
cd obsidian-calendar-plugin-remastered
npm install
npm run build
```

Скопируйте `main.js`, `manifest.json` и `styles.css` в `.obsidian/plugins/calendar/` вашего хранилища.

## ⚙️ Настройки

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| 📅 **Начало недели** | День начала недели | Системный |
| ✅ **Трекер задач** | Показывать/скрывать панель задач | Включено |
| 🎯 **Трекер привычек** | Показывать/скрывать панель привычек | Включено |
| 🗄️ **Архивация заметок** | Перемещать завершённые заметки-задачи в архив | Выключено |
| 📁 **Папка архива** | Папка для архивации (с поиском) | `Archive` |
| 🔄 **Синхронизация в хранилище** | Сохранять `calendar-data.json` в корне хранилища | Включено |
| 🔔 **Уведомления** | Включить напоминания о задачах | Выключено |
| ⏰ **За сколько минут** | Напоминать до начала задачи | 15 минут |

## ⌨️ Команды

Откройте палитру команд (`Ctrl/Cmd + P`) для быстрого доступа:

| Команда | Описание |
|---------|----------|
| 🕐 `Открыть расписание` | Открывает вкладку с FullCalendar |
| 📊 `Открыть аналитику привычек` | Открывает панель аналитики |
| 📍 `Reveal active note` | Показывает текущую заметку в календаре |
| 📝 `Open Weekly Note` | Создаёт/открывает еженедельную заметку |
