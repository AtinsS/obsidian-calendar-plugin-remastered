# 📅 Calendar Plugin Remastered

> **Мощнейший таск-трекер теперь и в Obsidian!** календарь, трекер задач и привычек, финансовый планировщик и аналитика — в одном плагине.

Разработан на основе [Calendar](https://github.com/liamcain/obsidian-calendar-plugin) от Liam Cain, но значительно расширен для полноценного управления жизнью и рабочими процессами.

## 🚀 Ключевые возможности

### 📆 Визуальное планирование
- **Основной календарь:** Индикаторы серий выполнения задач и привычек прямо в ячейках дат. Цели месяца визуально связаны с финансовым планированием.
- **Продвинутое расписание (FullCalendar):** Дневной, недельный и месячный виды. Создание задач кликом, drag & drop для изменения времени, цветовая кодировка проектов и адаптивная шкала времени для мобильных устройств.

### ✅ Умный трекер задач (Mini-Kanban)
Четыре статуса: *Сделать* → *В работе* → *На паузе* → *Готово*.
- **Гибкая привязка к заметкам:** 
  - *Обычная задача* со ссылкой на существующую заметку.
  - *Задача-заметка:* автоматическое создание `.md` файла (архивируется при завершении, удаляется при удалении задачи).
- **Тайм-менеджмент:** Встроенный таймер с логами. **Авто-возобновление** при перезапуске Obsidian (таймер продолжит тикать с того же места).
- **Автоматизация:** Повторяющиеся задачи (ежедневно/еженедельно/ежемесячно) и автоматическая очистка списка "Готово" для экономии памяти.
- **Приоритеты и дедлайны:** Цветовая маркировка проектов и сортировка.

### 🔁 Трекер привычек
- Три типа частоты: ежедневно, по выбранным дням недели (Пн-Вс), ежемесячно (конкретное число).
- Кастомизация иконок и цветов.
- Автоматический подсчёт и визуализация текущих и лучших серий (streaks).

### 💼 Финансы и Аналитика
- **Учёт рабочих задач:** Пометка задач как "Рабочие", расчёт заработка (почасовой или повременный), учёт переработок. Бейдж "💼" в интерфейсе.
- **Финансовый планировщик:** Общий баланс, основные счета, цели на месяц (отображаются в календаре для мотивации), правила распределения и предупреждения о превышении бюджета.
- **Дашборд аналитики:** График активности за 12 недель, визуализация логов времени, статистика привычек и помесячный график заработка.

## ⚙️ Технические особенности и Синхронизация

- **Хранение данных:** Все данные централизованно хранятся в файле `calendar-data.json` в корне хранилища.
- **Синхронизация:** Полная совместимость с **Obsidian Sync** и сторонними решениями (**Remotely Save**, **WebDAV**). 
- **Уведомления:** Нативные системные напоминания (от 1 до 60 минут до начала) и оповещения о просроченных дедлайнах.

## 📦 Установка

### Через BRAT (Рекомендуемый способ)
1. Установите плагин [BRAT](https://github.com/TfTHacker/obsidian42-brat).
2. Откройте настройки BRAT → **Add Beta Plugin**.
3. Вставьте URL репозитория: `https://github.com/AtinsS/obsidian-calendar-plugin-remastered`

### Вручную
1. Перейдите в раздел [Releases](https://github.com/AtinsS/obsidian-calendar-plugin-remastered/releases).
2. Скачайте `main.js`, `manifest.json` и `styles.css`.
3. Поместите их в папку `.obsidian/plugins/calendar-remastered/` внутри вашего хранилища Obsidian.
4. Включите плагин в настройках Obsidian.

## 🛠 Настройки по умолчанию

| Параметр | Значение | Описание |
|----------|----------|----------|
| Первый день недели | Как в системе | Зависит от локали ОС |
| Трекер задач / привычек | Включено | Базовая функциональность |
| Архивация завершённых задач | Выкл. | Автоматическое перемещение в папку архива |
| Папка для архива | `Archive` | Относительный путь в хранилище |
| Уведомления | Выкл. | Требует разрешения от ОС |
| Напоминание за | 15 минут | Время до начала события |
| Тип оплаты по умолчанию | В час | Для расчёта рабочих задач |
| Ставка по умолчанию | 0 ₽ | Базовая ставка для аналитики |

## 🧰 Технологический стек

Плагин разработан с упором на производительность и типизацию:
- **UI:** [Svelte 3.x](https://svelte.dev/) (лёгкий и реактивный, идеален для Obsidian)
- **Календарь:** [FullCalendar 6.x](https://fullcalendar.io/)
- **Работа с датой:** [Luxon 3.x](https://moment.github.io/luxon/)
- **Язык:** [TypeScript 4.x](https://www.typescriptlang.org/)
- **Тестирование:** [Jest](https://jestjs.io/)
- **Графики:** Canvas API
