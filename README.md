# Calendar Plugin Remastered

Форк [obsidian-calendar-plugin](https://github.com/liamcain/obsidian-calendar-plugin) с трекером задач (мини-канбан), привычек, таймером и синхронизацией.

## Календарь

- Переход по ежедневным заметкам, создание заметок для дней без файла
- Бейдж задач на каждой ячейке: количество / просроченные / все выполнены
- Бейдж привычек: количество / 🏆 все выполнены за день
- Подсветка текущего дня и выбранной даты
- Tooltip с незавершёнными задачами при наведении
- Еженедельные заметки с номером недели

## Мини-канбан

Три вкладки с задачами: **Сделать** / **В работе** / **Готово**.

- Статус задачи меняется через иконку слева (клик = цикл todo → progress → done)
- Быстрые кнопки [⏳] при наведении для перевода в «В работу»
- Цветовая полоска слева: серая (todo), оранжевая (progress), зелёная (done)
- Автоматическая миграция старых задач (completed → status)

## Таймер задач

- Автозапуск при переводе в «В работе», автостоп при «Готово»
- Отображение текущего времени в карточке задачи
- Логи времени сгруппированы по датам (кнопка ⏱ в шапке панели)
- Форматирование: «45 мин», «1 ч 20 мин»

## Временные оценки

- Поле «Ожидаемое время» при создании задачи (часы + минуты)
- Сравнение план/факт в карточке: зелёный если уложился, красный если переработал
- Пример: `⏱ 30 мин → ✔ 45 мин`

## Задачи по времени (Time-blocking)

- Назначение конкретного времени выполнения (например, 14:30)
- В колонке «Сделать» задачи со временем автоматически поднимаются вверх
- Сортировка по возрастанию времени
- Визуальное выделение: 🕒 и время в карточке
- Если время прошло — красная иконка ⚠️

## Задачи-заметки

- Задачи с привязкой к .md файлам отображаются с эмодзи 📝 вместо чекбокса
- Клик по 📝 открывает связанную заметку
- Визуальное выделение: фиолетовый border-left

## Трекер привычек

- Создание, редактирование, удаление привычек
- Частота: ежедневно / еженедельно / произвольные дни
- Отслеживание стриков, 🏆 трофей при выполнении всех за день

## Синхронизация

Данные хранятся в `calendar-data.json` в корне хранилища.

| Сервис | Настройка |
|--------|-----------|
| Obsidian Sync | Синхронизация файлов включена — файл подхватывается автоматически |
| Remotely Save | Добавьте `calendar-data.json` в **Regex of Paths to Allow** |
| iCloud / Google Drive | Хранилище синхронизируется полностью |
| Git | Файл захватывается автоматически |

В настройках плагина: **Синхронизация в корень хранилища** — вкл/выкл.

## Установка

### Через Community Plugins
1. Obsidian → Настройки → Community Plugins
2. Найти «Calendar», установить и включить

### Ручная установка
Скопируйте папку `calendar/` в `.obsidian/plugins/`:
```
calendar/
├── manifest.json
├── main.js
└── styles.css
```

## Настройки

| Раздел | Параметр | Описание |
|--------|----------|----------|
| Основные | Начало недели | День начала недели (locale / пн-вс) |
| Основные | Слов на точку | Количество слов для одной точки (250) |
| Основные | Подтверждение | Модальное окно перед созданием заметки |
| Недельные заметки | Номер недели | Столбец с номером недели |
| Задачи | Показывать трекер | Вкл/выкл панель задач |
| Задачи | Архивировать | Перемещение в папку архива |
| Привычки | Показывать трекер | Вкл/выкл панель привычек |
| Синхронизация | В корень хранилища | Сохранять в calendar-data.json |

## Горячие клавиши

| Команда | Действие |
|---------|----------|
| Calendar: Open view | Открыть календарь |
| Calendar: Open weekly note | Открыть недельную заметку |
| Calendar: Reveal active note | Показать активную заметку на календаре |

## Совместимость

Obsidian v0.9.11 или выше. Svelte 3, TypeScript.
