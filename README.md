# Calendar Plugin Remastered

Календарь, трекер задач и трекер привычек для [Obsidian](https://obsidian.md). Модифицированная версия оригинального плагина [Calendar](https://github.com/liamcain/obsidian-calendar-plugin) от Liam Cain.

## Возможности

### Календарь

- Месячный вид с навигацией по неделям
- Еженедельные заметки (опционально)
- Цветовые индикаторы задач и привычек на ячейках календаря
- Бейджи с количеством незавершённых задач
- Индикатор серии (streak) для ежедневных активностей
- Подсчёт слов как визуальный метрик
- Поддержка локали и настройка дня начала недели
- Glassmorphism-дизайн с адаптивной вёрсткой

### Расписание (FullCalendar)

- Недельный, дневной и месячный виды
- Time Grid с отображением задач по времени
- Drag & Drop для переноса и изменения длительности задач
- Swipe-навигация на мобильных устройствах
- Цветовая кодировка по проектам и статусам
- Создание задач по клику на пустое время

### Трекер задач

- Три статуса: **Сделать**, **В работе**, **Готово** (Kanban-стиль)
- Проекты с цветовой маркировкой
- Приоритеты: низкий, средний, высокий
- Повторяющиеся задачи (ежедневно, еженедельно, ежемесячно, по дням недели)
- Привязка задач к заметкам в хранилище
- Таймер отслеживания времени ( plan → fact)
- Оценка времени и сравнение с фактическими затратами
- Запланированное время с индикатором просроченности
- Описание задач с всплывающим окном
- Архивация заметок-задач при завершении
- Фильтрация по проектам
- Логи времени с историей за 30 последних записей

### Трекер привычек

- Ежедневные, еженедельные и пользовательские частоты
- Иконки и цвета для каждой привычки
- Подсчёт серии (streak)
- Отслеживание выполнения по датам
- Привязка к проектам

### Синхронизация

- Сохранение данных в `calendar-data.json`
- Синхронизация через Obsidian Sync, iCloud, Git, Remotely Save
- Автоматическая перезагрузка при изменениях файла

## Установка

### Ручная установка

1. Скачайте `main.js`, `manifest.json` и `styles.css` из последнего релиза
2. Скопируйте файлы в `.obsidian/plugins/calendar/`
3. Включите плагин в настройках Obsidian

### Через Obsidian BRAT

1. Установите плагин [BRAT](https://github.com/TfTHacker/obsidian42-brat)
2. Добавьте URL репозитория в настройках BRAT

## Настройки

| Настройка | Описание |
|-----------|----------|
| Начало недели | День начала недели (по умолчанию — системный) |
| Трекер задач | Показывать/скрывать панель задач |
| Трекер привычек | Показывать/скрывать панель привычек |
| Архивация заметок | Перемещать завершённые заметки-задачи в архив |
| Папка архива | Папка для архивации (по умолчанию `Archive`) |
| Синхронизация в хранилище | Сохранять `calendar-data.json` в корне хранилища |

## Команды

| Команда | Описание |
|---------|----------|
| `Открыть расписание` | Открывает вкладку с FullCalendar |
| `Reveal active note` | Показывает текущую заметку в календаре |
| `Open Weekly Note` | Создаёт/открывает еженедельную заметку |


## Требования

- Obsidian ≥ 0.9.11
- Рекомендуется плагин [Daily Notes](https://help.obsidian.md/Plugins/Daily+notes) или [Periodic Notes](https://github.com/liamcain/obsidian-periodic-notes)

## Технологии

- [Svelte](https://svelte.dev/) 3.x
- [FullCalendar](https://fullcalendar.io/) 6.x
- [Luxon](https://moment.github.io/luxon/) для работы с датами
- TypeScript
