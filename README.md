# 🗓️ Calendar Plugin Remastered

> Всё в одном: умный календарь, трекер задач и привычек, тайм-менеджмент и финансовый планировщик для Obsidian.

Плагин расширяет возможности классического [obsidian-calendar-plugin](https://github.com/liamcain/obsidian-calendar-plugin), добавляя мощный движок расписания на базе библиотеки **FullCalendar** и полноценный финансовый учёт.

## ⚡ Чем этот плагин выделяется

- **Один для всего:** Не нужно переключаться между 5 разными плагинами для календаря, задач, учёта времени работы и финансов.
- **Полная совместимость:** Данные хранятся в прозрачном JSON-формате и легко синхронизируются через Obsidian Git, Remotely Save или WebDAV (Яндекс.Диск, OneDrive).
- **Полноценные уведомления в Obsidian**: можно настроить нативные уведомления windows, либо через [ntfy.sh](https://ntfy.sh/) (можно скачать на смартфон) уведомления к задачам могут приходить даже когда компьютер выключен
- **Синхронизация с заметками:** Интеграция с плагином [Tasks](https://github.com/obsidian-tasks-group/obsidian-tasks) и Dataview через обычные `.md` файлы.

## 🚀 Возможности

### 📅 Календарь и расписание
- Визуальные индикаторы задач и привычек прямо в сетке календаря.
- Продвинутые виды расписания благодаря библиотеке FullCalendar.
- Создание задач кликом и изменение времени через drag & drop в расписании.

### ✅ Задачи и тайм-менеджмент 
- **4 статуса:** *Сделать* → *В работе* → *На паузе* → *Готово*.
- **Гибкость:** Повторяющиеся задачи, привязка к проектам и `.md` заметкам.
- **Таймер:** Встроенный учёт времени с автовозобновлением при перезапуске Obsidian (рекомендую установить плагин Tray, для удобства)
- **Контроль:** Уведомления о дедлайнах и превышении запланированного времени.

### 🔄 Трекер привычек
- Гибкая периодичность (ежедневно / еженедельно / ежемесячно) с количественными целями.
- Подсчёт стриков (серий) и аналитика продуктивности по дням недели.

### 💰 Финансы и аналитика
- **Доходы:** Расчёт стоимости рабочих задач (почасовая / подённая ставка) с учётом переработок.
- **Бюджет:** Месячные категории расходов, цели накопления.
- **Автоматизация:** Правила распределения остатков и перенос трат с прошлого месяца.
- **Графики:** Визуализация аналитики заработка и расходов.

## 🔄 Синхронизация

### Новый подход к синхронизации между устройствами
Плагин хранит данные в `calendar-data.json`. Вы можете выбрать хранение в корне хранилища  для бесшовной работы с:
- **Obsidian Sync**
- **Remotely Save**
- **WebDAV** (Яндекс.Диск, OneDrive и др.)

### Внешние календари (Google Calendar, Apple Calendar и др.)
1. Создайте [GitHub Personal Access Token](https://github.com/settings/tokens) (классический) со скопом `gist`.
2. Вставьте токен в настройки плагина и нажмите **«Синхронизировать»**.
3. Плагин создаст Gist с `.ics` файлом и выдаст ссылку.
4. Добавьте эту ссылку в свой календарь через функцию «Подписка по URL».

### Интеграция с форматом Tasks
При включённой настройке плагин создаёт `.md` файлы для задач, обеспечивая двустороннюю синхронизацию с плагином **Tasks** и **Dataview**.

```markdown
---
task_id: abc123
title: Купить молоко
status: todo
date: 2024-10-25
priority: medium
---
- [ ] Купить молоко 📅 2024-10-25 ⏰ 14:30
```
*(Поддерживаемые статусы: `- [ ]`, `- [/]`, `- [-]`, `- [x]`)*

## 🔔 Уведомления и автоматизация

### Локальные уведомления

Настраиваются индивидуально для каждого типа:

| Тип | Когда срабатывает |
| :--- | :--- |
| **Напоминания** | За N минут до начала задачи. |
| **Просроченные** | Мгновенно, если задача не начата в срок. |
| **Лимит времени** | При превышении запланированной оценки. |
| **Дедлайны** | За день, в день и в момент истечения срока. |

### ☁️ Уведомления при выключенном ПК (через GitHub Actions)
Плагин может отправлять утренние сводки (эксперементально) и алерты на телефон через [ntfy.sh](https://ntfy.sh/) (мобильное приложение), даже когда Obsidian закрыт.

**Как настроить (3 шага):**
1. Включите в настройках плагина **«Синхронизация в корень хранилища»**.
2. Скопируйте папку `examples` из репозитория плагина в корень вашего хранилища и переименуйте её в `.github`.
3. Создайте [GitHub Token](https://github.com/settings/tokens) с правами `repo` и `actions:write`, и вставьте его в настройки плагина.
> *Требуется настроенный автоматический `git push` (например, через плагин Obsidian Git).*

## 📦 Установка

### Через BRAT (Рекомендуется)
1. Установите плагин [BRAT](https://github.com/TfTHacker/obsidian42-brat).
2. Откройте настройки BRAT → **Add Beta Plugin**.
3. Вставьте ссылку: `https://github.com/AtinsS/obsidian-calendar-plugin-remastered`
4. Нажмите **Add Plugin**.

### Вручную
1. Скачайте архив, либо клонируйте, выделите `main.js`, `manifest.json` и `styles.css`
2. Поместите их в папку `.obsidian/plugins/calendar-plugin-remastered/` (создайте её, если нет).
3. Включите плагин в *Настройки → Сторонние плагины*.
