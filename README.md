# Calendar Plugin Remastered

> Календарь, трекер задач, трекер привычек, финансовый планировщик и аналитика — в одном плагине для Obsidian.

Разработан на основе [Calendar](https://github.com/liamcain/obsidian-calendar-plugin) от Liam Cain, но значительно расширен для полноценного управления задачами, временем и финансами.

## Возможности

### Визуальное планирование
- **Основной календарь** — индикаторы задач и привычек прямо в ячейках дат.
- **Продвинутое расписание (FullCalendar)** — дневной, недельный и месячный виды, создание задач кликом, drag & drop для изменения времени.

### Трекер задач
Четыре статуса: *Сделать* → *В работе* → *На паузе* → *Готово*.
- Повторяющиеся задачи (ежедневно / еженедельно / ежемесячно).
- Автоматическая очистка выполненных задач (настраиваемый лимит).
- Дедлайны с уведомлениями: за день, в день дедлайна, при истечении времени.
- Оценка времени с автоматическим уведомлением при превышении лимита.

#### Синхронизация с заметками (формат Tasks)
- **Автоматическое создание** — при включённой настройке каждая задача получает `.md` файл в формате, совместимом с плагином Tasks.
- **Двусторонняя синхронизация** — изменения в заметке обновляют JSON-хранилище плагина и наоборот.
- **Привязка существующих заметок** — любую `.md` заметку можно вручную привязать к задаче.

Формат заметки:
```markdown
---
task_id: abc123
title: Купить молоко
status: todo
date: 2024-10-25
priority: medium
---

- [ ] Купить молоко 📅 2024-10-25 ⏰ 14:30
```
*Статусы:* `- [ ]` (todo), `- [/]` (progress), `- [-]` (paused), `- [x]` (done).

#### Тайм-менеджмент
- Встроенный таймер с логами (макс. 30 записей).
- Авто-возобновление при перезапуске Obsidian.
- Накопленное время сохраняется между сессиями.

#### Рабочие задачи и финансы
- Тип оплаты: за час / за день.
- Настраиваемая ставка и порог overtime (сверхурочных).
- Расчёт стоимости по задачам, за месяц и за год.

### Трекер привычек
- Ежедневные и недельные привычки.
- Мульти-таргет (цель в количественном выражении).
- Стрики (серии) и аналитика.
- Тепловая карта на календаре.

### Финансовый планировщик
- Месячные бюджеты по категориям.
- Цели накопления.
- Правила распределения средств.
- Аналитика: расходы, доходы, тренды.

## Уведомления

Плагин поддерживает несколько типов уведомлений, каждый из которых можно включить или выключить отдельно:

| Тип | Описание |
|-----|----------|
| **Напоминания** | За N минут до запланированного времени. |
| **Просроченные задачи** | Сразу при наступлении запланированного времени, если задача всё ещё в статусе "todo". |
| **Превышение лимита времени** | Когда фактическое время работы превышает заявленную оценку. |
| **Дедлайны** | За день до дедлайна, в день дедлайна, при истечении времени. |
| **Утренняя сводка** | Отправка сводки через ntfy.sh  по расписанию или по текстовой команде с интеграцией Github Actions («сводка» / «утренняя сводка» с ответом в течение 30 сек). |

> 💡 **Дополнительно:** Полная интеграция с [ntfy.sh](https://ntfy.sh/) для отправки push-уведомлений на смартфон.

## Синхронизация задач с заметками (работа с плагином Tasks)

| Настройка / Действие | Описание |
|-----------|----------|
| **Создавать Task-заметку для каждой задачи** | Автоматически создавать `.md` файл в папке задач. |
| **Папка для задач** | Путь для хранения Task-заметок (по умолчанию `Tasks/`). |
| **Лимит выполненных задач** | Максимальное количество хранимых выполненных задач (по умолчанию 180, далее следует автоочистка). |
| **Создать заметки для всех задач** | Кнопка для массового создания Task-заметок из существующего JSON-хранилища. |

> 💡Можно интегрировать с DATAVIEW, Telegeam Reminder, Kanban и многое другое...

## Синхронизация данных

| Настройка | Описание |
|-----------|----------|
| **Синхронизация в корень хранилища** | Сохранять данные в `calendar-data.json` в корне хранилища вместо папки плагина (рекомендуется для совместной работы). |

**Поддерживаемые способы синхронизации:**
- **Remotely Save** — добавьте `calendar-data.json` в "Include Files".
- **Obsidian Sync** — нативная синхронизация файлов хранилища.
- **iCloud / Google Drive / Яндекс.Диск** — полная синхронизация папки хранилища через облако.

## Установка

### Через BRAT (рекомендуется)
1. Установите плагин [BRAT](https://github.com/TfTHacker/obsidian42-brat).
2. Откройте настройки BRAT → **Add Beta Plugin**.
3. Вставьте URL: `https://github.com/AtinsS/obsidian-calendar-plugin-remastered`

### Вручную
1. Скачайте `main.js`, `manifest.json` и `styles.css` из раздела [Releases](https://github.com/AtinsS/obsidian-calendar-plugin-remastered/releases).
2. Поместите их в папку `.obsidian/plugins/calendar-plugin-remastered/` внутри вашего хранилища Obsidian.
3. Включите плагин в настройках Obsidian (Settings → Community plugins).

> Для использования уведомлений Github Actions скопируйте папку `.github` в корень хранилища.
